# QuizMaster

QuizMaster is an engaging quiz application that challenges users with multiple-choice questions. It features a timer, score tracking, and a leaderboard. The application is built with HTML, CSS, and JavaScript, and is designed to be responsive and easy to use.

## Features

- **Interactive Quiz**: Answer multiple-choice questions and receive immediate feedback.
- **Timer**: 30-second timer for each question to add urgency.
- **Score Tracking**: Tracks and displays your score based on correct answers.
- **Leaderboard**: Shows the top scores from all users.
- **Progress Bar**: Visual indicator of quiz progress.
- **Responsive Design**: Optimized for both desktop and mobile devices.

## Installation

To run QuizMaster locally, follow these steps:

1. **Clone the repository:**

    ```bash
    git clone https://github.com/your-username/quizmaster.git
    ```

2. **Navigate to the project directory:**

    ```bash
    cd quizmaster
    ```

3. **Open `index.html` in your web browser.**

    You can open the file directly in your browser or use a local server for a better development experience.

## Usage

1. **Start the Quiz:**
   - Open `index.html` in your browser.
   - Enter your name and click "Start Quiz" to begin.

2. **Answer Questions:**
   - Choose your answer by clicking one of the buttons.
   - Click "Next" to move to the next question.

3. **View Results:**
   - After completing the quiz, your score will be displayed.
   - The leaderboard will show your rank.

4. **Restart the Quiz:**
   - Click "Restart Quiz" to play again.
   - Use the navigation links to view the rules or return to the home page.

## Files

- **`index.html`**: The main quiz page.
- **`rules.html`**: Page outlining quiz rules.
- **`script.js`**: Contains the JavaScript logic for quiz functionality.
- **`styles.css`**: Styling for the quiz application.
